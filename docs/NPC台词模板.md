# NPC 答题气泡台词模板

这个文件用于设计每个 NPC 在答题时弹出的小对话框。

## 填写规则

每个 NPC 有两类台词：

- `win`：范宇轩答错时，NPC 得势、嘲讽、压迫的话。
- `wrong`：范宇轩答对时，NPC 吃瘪、认栽、嘴硬的话。

建议每类至少写 3 句。游戏里会按顺序循环显示。

## 代码填写位置

实际游戏读取的文件在：

```text
src/data/npcReactionLines.js
```

你可以把下面模板里的句子改好后，再让我帮你填进代码。

## 写台词建议

每句尽量控制在 8-16 个字，太长会挤爆气泡。

推荐风格：

- 有朋友梗
- 轻松嘲讽
- 不攻击现实人格
- 不低俗
- 不涉及隐私
- 能让当事人看了会笑

示例：

```text
这题都能错？材料又多一页。
这波澄清有点东西。
你被第一层骗到了。
今天人脉也救不了这题。
```
